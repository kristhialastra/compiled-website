Dive deep and reflect ~

As a freshman student with no IT background or prior experience in web development, my experience in the Web Systems Technology subject has been both challenging and exciting. During this semester, I have been introduced to the basic concepts of web development, such as HTML, CSS, and JavaScript, and I have learned how to build a website from scratch.

One of the most significant challenges I faced was understanding the various programming languages and technologies involved in web development. At the beginning, I felt overwhelmed by the sheer number of technologies we were expected to learn, from HTML and CSS to JavaScript and PHP. To overcome this challenge, I took a proactive approach to my learning by seeking out additional resources outside of the classroom. I watched online tutorials, read books and articles.

Building a website for the first time was a daunting task, but also a fulfilling experience. One of the biggest challenges I faced was learning how to debug errors in my code. When I encountered an error in my code, I often felt overwhelmed and didn't know where to start looking for the problem. This was particularly frustrating when I thought I had written the code correctly, but the website was not displaying the expected results.

To overcome this challenge, I learned to break down my code into smaller, more manageable pieces and test each component individually. I also made use of debugging tools like the Debug CSS chrome extension and online resources to help me identify and resolve issues in my code. Through this process, I gained a better understanding of how code works and how to write more efficient and effective code.

Of course, there have been other challenges along the way (that if I discuss in details will be like an episode of MMK), but with the help of my professor and classmates, I feel like I've been able to overcome these challenges and grow as an IT student.

As we move forward in the semester, I'm excited to continue building on what I've learned so far. I feel like I've just scratched the surface of what's possible, and I'm eager to explore more about web development. All in all, I'm really glad we have this subject and I feel like it's helped me like my course a bit more, and decreased my desire to shift.
